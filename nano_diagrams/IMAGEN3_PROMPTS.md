## 使用说明

### 方法1: 使用Imagen 3 CLI (如果可用)
```bash
cd /d/cowork/ppt/nano_diagrams
imagen generate "$(cat architecture_prompt.txt)" --output architecture_pyramid.png --size 1920x1080
```

### 方法2: 使用Gemini CLI with Imagen
```bash
gemini imagen "提示词内容" -o output.png
```

### 方法3: 使用Google AI Studio
1. 访问 https://aistudio.google.com/
2. 选择 Imagen 3 模型
3. 粘贴提示词
4. 生成并下载图片

### 方法4: 备用Python生成
如果Imagen 3不可用，使用Python matplotlib生成：
```bash
python generate_architecture_diagram.py
python generate_comparison_chart.py
# ... 其他图表生成脚本
```
